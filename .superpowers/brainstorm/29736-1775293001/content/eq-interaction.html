A — Drag dots directly on the curve ★ Expert pick

Each band is a draggable dot sitting on the frequency curve. Drag up/down = gain. The curve redraws in real time. Clicking a dot shows a tooltip with the exact dB value. This is the Fabfilter / Neutron paradigm — used by every modern pro EQ. Compact: the entire EQ fits in one panel without vertical sliders below.

Pros

  • Interaction IS the visualisation — no separate controls
  • Very compact — fits the bottom drawer perfectly
  • Direct manipulation feels fast and professional

Cons

  • No separate precise numeric input (mitigated by tooltip + dblclick to reset)

B — Curve (read-only) + vertical sliders below

Curve is a passive display — you adjust gains via 10 vertical sliders underneath it. Two separate areas doing the same job. Takes more vertical space, so the bottom drawer needs to be taller. The slider thumb is also a smaller hit target than a dot on the curve.

Pros

  • Familiar pattern for users who know graphic EQs
  • Clear separation between display and controls

Cons

  • Taller panel needed — more screen covered
  • Redundant — curve and sliders show the same information twice
