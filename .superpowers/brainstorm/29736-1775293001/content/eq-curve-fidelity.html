A — Full frequency curve ★ Expert pick

Smooth SVG bezier curve computed from all 10 bands using the Web Audio BiquadFilter frequency response API (getFrequencyResponse()). Filled area below 0dB line. Draggable band dots on the curve. Redraws every frame while dragging. This is what every serious EQ (foobar2000, Fabfilter, Neutron) shows.

Pros

  • Accurate — uses actual browser filter math
  • Instantly communicates the combined effect
  • Drag directly on the curve = intuitive

Cons

  • Requires computing 512 frequency points per frame while dragging

B — Vertical sliders only (no curve)

10 vertical gain sliders, no frequency response overlay. Simpler to build. The missing piece: you can't see how adjacent bands interact — a +12dB at 1kHz bleeds heavily into adjacent bands, and you'd have no visual feedback of that.

Pros

  • Simpler implementation
  • Familiar for basic EQ apps

Cons

  • No feedback on combined frequency shape
  • Feels like a phone app, not a pro tool
